Technical Projects

Created and developed programs during Software Engineering Certification at Flatiron School


PlayBall - GitHub

Created app that allows users to find parks in their area that have the proper accommodations to play either baseball, tennis, basketball or soccer. The user also has the ability to create PickUps(events) that other users can RSVP to attend.

  • Utilized Ruby on Rails/PostgreSQL/SQL and Active Record to build a backend API that allows the user to store data.
  • Incorporated Bcrypt to store and encrypt login information creating a secure browsing experience for users.
  • Utilized Restful routes to fetch data via a JSON server streamlining the sending of user data to the database.

MovieMania - GitHub
